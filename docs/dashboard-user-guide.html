Daily quick start

Seven steps from sign-in to completed route.

  1. Sign inOpen the admin dashboard with an approved administrator account.
  2. Review New requestsSelect View and confirm contact details, address, plan, bin count, and requested date.
  3. Schedule serviceSet the next pickup date, choose the right status, add private notes, and save.
  4. Check Home BaseMake sure the operator’s starting address and return-to-base preference are correct.
  5. Build the routeChoose the date, select up to 25 eligible pickups, and select Build best route map.
  6. Verify and navigateCheck the numbered order, names, and addresses; then select Start navigation.
  7. Close out the dayMark one-time work Completed or schedule the next date for Active customers.

Manage customer pickups

Review, update, search, and safely remove records.

🔎 Find a customer

Search by name, email, phone, street address, or ZIP code. Select Refresh to load the newest records.

✏️ Update a request

Select View, change the pickup date or status, add private admin notes, then select Save changes.

🗑️ Delete a pickup

Use only for a duplicate, test, or record that truly must be removed. Deletion is permanent.

Permanent means permanent. Read the customer name in the confirmation message before deleting. A deleted pickup cannot be restored from the dashboard.

Set Home Base

The private start—and optional finish—of every optimized route.

  1. Open Home BaseUse the dashboard header or left menu.
  2. Enter operator detailsAdd name, phone, street address, city, state, and ZIP code.
  3. Choose the return settingDecide whether the route should return to Home Base after the last pickup.
  4. Save Home BaseThe route planner will use it for future routes.

Recommended operating routine

A simple rhythm for accurate records and smoother pickup days.

🌅 Start of day

Refresh, review New requests, confirm today’s pickups, verify Home Base, and build the route.

🚚 During the route

Verify each address, follow the numbered order, and record access issues in private notes.

🌙 End of day

Complete finished pickups, schedule recurring customers, resolve missed stops, and sign out.

Troubleshooting

Fast checks for the most common dashboard problems.

A pickup does not appear in the route

Confirm its status is Scheduled or Active and its pickup date exactly matches the date selected in Route Planner.

The route button is disabled

Set Home Base, choose a date with eligible pickups, and make sure at least one stop is checked.

The embedded map does not load

Use Start navigation if it is available. An administrator should verify the Google Maps JavaScript API, Routes API, and Maps Static API settings in Google Cloud and Netlify.

The dashboard says waypoint optimization

The route is still optimized and usable. This is the automatic fallback used when the advanced Route Optimization service is not configured or temporarily unavailable.

Requests will not load or save

Refresh and sign in again. If it continues, record the complete browser-console message and the time it occurred.

Login is rejected

Confirm the email and password. The account must also be on the approved administrator list.

Security + handoff checklist

Protect customer information and train new operators safely.

  • Use only trashgrab.app/admin.html for live administration.
  • Never share API keys, passwords, Supabase keys, or login tokens.
  • Give access only to approved staff.
  • Never put passwords in customer or admin notes.
  • Sign out on shared devices.
  • Verify each customer address before driving.
  • Explain that deletion cannot be undone.
  • Verify Home Base and the return setting.
  • Create and route a test pickup for training.
  • Delete the test record after training.
